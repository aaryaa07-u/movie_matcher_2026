# Movie Matcher App

A Python Flask web application for discovering movies using real IMDb data with advanced search, authentication, and personalized recommendations.
Movie Matcher is a movie recommendation platform that helps users decide what to watch next by curating suggestions based on community reviews. 
Users rate and review films they have watched, and these reviews are publicly visible, allowing others to read and compare opinions before deciding to watch a film.

## Features

- **Real IMDb Data**: Downloads and processes official IMDb datasets
- **Advanced Search**: Search by title, genre, year, cast, and rating
- **User Authentication**: Secure register and login system
- **Movie Details**: Comprehensive movie information pages
- **Ratings**: Users will answer a 3 question survey to determine their percentage rating of their selected film
- **Recommendations**: Smart filtering by ratings and popularity
- **Responsive UI**: Modern Bootstrap interface
